# 💰 Finance Dashboard

A clean, responsive, and interactive **Finance Dashboard** built using **React, Bootstrap, and JavaScript** to help users track and analyze their financial activity.

---

## 📌 Features

### 📊 Dashboard Overview

* Total Balance, Income, and Expense summary cards
* Time-based trend visualization (Line Chart)
* Category-based breakdown (Pie Chart)

### 📋 Transactions Management

* View all transactions with details (Date, Amount, Category, Type)
* Add, Edit, and Delete transactions
* Search, filter, and sort functionality

### 🔐 Role-Based UI

* **Viewer** → Can only view data
* **Admin** → Can add, edit, and delete transactions

### 💡 Smart Insights

* Highest spending category
* Weekly spending comparison
* Balance trend insights

### 📤 Export Functionality

* Export transactions as **CSV**
* Export transactions as **JSON**

### 🎯 Advanced UI Features

* Hide/Show financial values
* Toast notifications for actions
* Responsive design (mobile, tablet, desktop)
* Dark / Light mode toggle

---

## 🛠️ Tech Stack

* ⚛️ React.js
* 🎨 Bootstrap + Custom CSS
* 📊 Recharts (for charts)
* 🔔 React Toastify

## ⚙️ Installation & Setup

1. Clone the repository:

```bash
git clone https://github.com/your-username/finance-dashboard.git
```

2. Navigate to project:

```bash
cd my-app
```

3. Install dependencies:

```bash
npm install
```

4. Start the app:

```bash
npm start
```
